# ChatBat

ChatBat is a web-based chat application built with **JavaScript**, **HTML**, and **CSS**, bundled using **Vite**. It provides a sleek interface for chatting and can be easily extended for additional functionality.

---

## Features

- Real-time chat interface
- Clean, responsive design
- Easy to extend and customize
- Built with modern front-end tooling (Vite)

---

## Getting Started

These instructions will help you run ChatBat locally on your machine.

### Prerequisites

Make sure you have **Node.js** installed. You can download it from [nodejs.org](https://nodejs.org/).

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/BakerMoney/ChatBat.git
    ```

2. Navigate to the project folder:

    ```bash
    cd ChatBat
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

### Running the Project

Start the development server:

```bash
npm run dev
```

Open your browser and go to the URL provided in the terminal (usually `http://localhost:5173`) to see the app in action.
